Message sending settings

Maximum size for chat messages:
Normally, it is 300 characters, but server administrators can bypass this limit.

Pokemon Showdown Message Throttle Delay (milliseconds):
Pokemon showdown has a restriction on how fast you can send messages.
For regular users, it is 600 milliseconds of delay. For global ranks or public room staff, it is 150 milliseconds of delay.
Set this value accordingly with the rank of your bot account.

Max size of message buffer:
Max number of messages to keep in queue if they cannot be sent due to the throttle delay.
