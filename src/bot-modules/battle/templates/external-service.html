An external battle bot service allows you to delegate the battle bot AI to an external service.
The bot calls this service by calling an API that must be implemented by the service. Check the API specification if you are planning to implement your own service. You can use the Swagger Editor to visualize the specification and generate a template server.

 Enable external battle bot service.

Base URL of the service: 
This URL contains the host and the port of the server hosting the battle bot service.

Service authorization: 
Authorization token for the bot to use the service.

Authorization header name: 
The name of the Authorization header for the HTTP requests to the service.
